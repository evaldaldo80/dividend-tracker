Bienvenue — démarrer en 3 étapes
1
Saisir tes premiers dividendes
Va dans l'onglet Données et ajoute tes paiements un par un via le formulaire — ou importe directement un fichier CSV si tu as un historique Excel.

Format CSV attendu (une ligne par paiement) :
date,ticker,montant
2024-01-15,JGPI,125.50
2024-01-20,YGLD,89.30
2024-02-15,JGPI,130.00
2024-03-01,O,47.20
Le bouton Template CSV te télécharge un fichier d'exemple prêt à remplir.
Si tu n'as que des totaux mensuels (pas le détail par ticker), utilise DIVERS comme ticker.
2
Sauvegarder tes données entre les sessions
Tes données sont automatiquement mémorisées dans ton navigateur (localStorage). Elles survivent à la fermeture de l'onglet — mais pas à la suppression du cache.

Pour une sauvegarde fiable, utilise le bouton Sauvegarder (.json) régulièrement. Ça télécharge un fichier dividendes.json sur ton PC. Pour restaurer sur un autre appareil ou après un reset navigateur, clique Restaurer (.json).
3
Héberger la page sur GitHub (gratuit, en 5 min)
GitHub Pages te permet d'accéder à ce tableau de bord depuis n'importe quel appareil via une URL personnelle — sans abonnement, sans serveur.
Étape 1 — Créer un compte sur github.com (gratuit)
Étape 2 — Cliquer sur + → New repository
    · Nom : dividend-tracker
    · Visibilité : Public (obligatoire pour Pages gratuit)
    · Cocher "Add a README file" → Create repository
Étape 3 — Cliquer sur Add file → Upload files
    · Glisser ce fichier HTML et le renommer index.html
    · Cliquer Commit changes
Étape 4 — Aller dans Settings → Pages
    · Source : Deploy from a branch → Branch: main → Save
Étape 5 — Attendre 2 min, ton URL sera :
    https://TON-USERNAME.github.io/dividend-tracker/
⚠️ Note : le localStorage est lié au navigateur et à l'URL. Une fois hébergé sur GitHub, tes données seront liées à l'URL GitHub — pas à ton PC local. Utilise Sauvegarder / Restaurer (.json) pour transférer tes données.
?
Questions fréquentes
Mes données sont-elles en sécurité ?
Oui — tout reste sur ton appareil. Ce fichier HTML ne communique avec aucun serveur extérieur. Tes chiffres ne quittent jamais ton navigateur.

Puis-je utiliser plusieurs devises ?
Le tracker affiche tout en €. Si tu as des dividendes en USD, convertis manuellement au moment de la saisie (ou importe le montant converti).

Comment mettre à jour le fichier sur GitHub ?
Va dans ton repo → clique sur index.html → icône crayon → colle le nouveau contenu → Commit changes. Tes données localStorage sont indépendantes du fichier HTML.
